## Core navigation

- `w term` for windows-first filtering
- `a brave` for apps-first filtering
- `f report` for indexed file matches

## Utility intents

- `emoji smile`
- `zurich time`
- `100usd to eur`
- `weather berlin`
- `29*58`

## Web actions

For non-empty queries, web search actions are appended to the end of the result list using configured providers.
